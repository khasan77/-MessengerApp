**1.UIKit**

Экраны авторизации, список чатов, диалог и профиль. Используются кастомные вьюшки, таблицы, Auto Layout и навигация через UINavigationController.

**2.Регистрация и работа с аккаунтом**

Реализована регистрация и авторизация через Firebase Authentication. Пользователь может выбрать фото при регистрации — изображение загружается в Firebase Storage и привязывается к аккаунту.

**3.Чаты и сообщения**

Приложение позволяет создавать новые чаты, закреплять важные чаты и удалять их.

**4.Работа с сетью**

Получение, отправка и обновление данных выполняются через Firebase (Auth, Database, Storage).

**5.Хранение и структура данных**

Пользовательские данные, чаты и сообщения хранятся в Firebase с разделением по узлам. Фото профиля лежит в Firebase Storage. Все операции чтения/записи оформлены в отдельные менеджеры.

**7.Архитектура проекта**

Логика вынесена в отдельные сервисы: работа с Auth, базой данных и Storage. Экраны построены через контроллеры и модели данных. Структура кода разделена так, чтобы легко добавлять новые функции.

Технологии и зависимости:

UIKit,
AutoLayout,
UserDefaults,
Firebase Authentication,
Firebase Database,
Firebase Storage,
MessageKit,
Auto Layout,
GCD,
MVC,
CocoaPods.
